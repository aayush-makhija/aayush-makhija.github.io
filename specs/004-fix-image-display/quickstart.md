# Quickstart: Verification Guide

## Post-Implementation Visual Checks

### 1. Color Verification
- Load the home page.
- Scroll through the **Hero** and **Projects** sections.
- **Expected Outcome**: All images should appear in full color without hovering.

### 2. Intro Image Framing
- In the **Hero** section, observe the profile photo.
- **Expected Outcome**: The subject's hair/head should be fully visible and not cut off by the top edge of the rounded box.

### 3. Responsive Framing
- Resize the browser window from desktop to mobile widths.
- Observe the profile photo during the transition.
- **Expected Outcome**: The image should stay centered and focus on the top part (hair/head) as the box dimensions change.

### 4. Interactive Effects
- Hover over the profile image and project cards.
- **Expected Outcome**: 
    - Profile: Tilt animation still works; no color change occurs.
    - Project Cards: Scale-up animation still works; no color change occurs.
