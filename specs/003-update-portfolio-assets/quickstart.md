# Quickstart: Update Portfolio Assets

## Setup
1. Ensure `photo.jpeg` is present in `src/assets/images/`.
2. Sourced `chatbot-preview.jpg` will be added to the same directory.

## Implementation
- Update `Hero.jsx` to import and use the local photo.
- Update `Projects.jsx` list to reference the local chatbot image.

## Verification
- Run `npm run dev` and inspect the Hero and Projects sections.
- Confirm hover transitions are working as expected.
