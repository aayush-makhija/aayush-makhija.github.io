# Quickstart: Project Documentation (README)

## Getting Started

Follow these steps to set up and run the Portfolio Website:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd website
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

5. **Lint the project**:
   ```bash
   npm run lint
   ```
