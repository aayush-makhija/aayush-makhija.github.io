# Quickstart: Enhanced UI & Content

## Prerequisites
- Node.js & npm installed
- Project dependencies installed via `npm install`

## Local Development
1. Run `npm run dev` to start the Vite dev server.
2. Open `http://localhost:5173` to view the portfolio.

## Applying Content Updates
- Real photos should be placed in `src/assets/images/`.
- Update text content in relevant sections: `src/sections/Hero.jsx`, `src/sections/Projects.jsx`, etc.

## Visual Verification
- Inspect the background for animated gradient blobs (FR-007).
- Test the mobile navigation overlay by resizing the browser to mobile width (FR-006).
- Verify animation behavior by scrolling through sections (FR-002).
