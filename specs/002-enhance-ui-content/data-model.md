# Data Model: Enhanced UI & Content

## Entities

### Developer Profile
- **Name**: String (e.g., Aayush Makhija)
- **Role**: String (e.g., Undergraduate Engineering Student)
- **Bio**: Text (personalized content from FR-004)
- **Social Links**: List of { name, url, icon }
- **Profile Photo**: Asset URL (real photo)

### Project
- **Title**: String
- **Description**: Text
- **Github URL**: URL
- **Screenshot**: Asset URL

### Experience Entry
- **Company**: String
- **Role**: String
- **Dates**: String
- **Achievements**: List of Text
