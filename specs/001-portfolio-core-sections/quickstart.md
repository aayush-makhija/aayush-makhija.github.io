# Quickstart: Portfolio Core Sections

## Project Setup
1. Clone the repository.
2. Open `index.html` in a modern web browser.
3. (Optional) Run a simple development server: `npx serve .`

## How to Add Content

### Update Projects
Modify the `My Work` section in `index.html`. Each project should have:
- Title
- Description
- Link to GitHub repository

### Update Skills
Add your technical skills to the `Skills` section. Group them using Tailwind CSS categories (e.g., `<div class="category">Frontend</div>`).

### Update Experience
Add your internships to the `Experience` section. Include:
- Company and Role
- Dates (Start - End)
- 3-4 achievement bullet points

### Update Current Status
Modify the `Current Commit` status card in `index.html` with your latest education and internship info.

### Update Contact Links
Update the href attributes in the `Contact Me` section icons and the resume download link.
