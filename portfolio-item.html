Slack Bot

Wolfram and Wit.ai Integration

This project is a Slack bot designed to answer user queries by leveraging the power of Wolfram Alpha and Wit.ai for natural language processing. Developed in Go (Golang), the bot listens to specific commands on Slack and provides intelligent, data-driven responses.

Key Features:

  • Wolfram Alpha Integration: Handles complex queries like factual questions, calculations, and more.
  • Wit.ai NLP: Understands natural language inputs to extract relevant entities and intents.
  • Real-Time Interaction: Uses Slackâ€™s API to provide instant responses in channels or direct messages.
  • Event Logging: Captures command events for easy debugging and analytics.

    • How It Works:
  • The bot listens for commands in the format q - message.
  • The input message is processed using Wit.ai to identify relevant entities.
  • The extracted query is sent to Wolfram Alpha for a precise, spoken answer.
  • The response is posted back in the Slack channel.

    • Technologies Used:
  • Go (Golang)
  • Slack API (via Slacker library)
  • Wit.ai for NLP
  • Wolfram Alpha API
  • GJSON for JSON parsing

    • This project showcases the power of combining AI-driven natural language understanding with robust APIs for creating smart, responsive Slack bots.
