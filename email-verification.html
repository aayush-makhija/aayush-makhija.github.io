Domain Verification

Golang

This tool is designed to verify the existence and email security configuration of a domain. It checks for MX, SPF, and DMARC records using DNS lookups to determine whether a domain is properly configured for email communication.

Key Features:

  • MX Record Lookup – Checks if the domain has valid mail exchange (MX) records.
  • SPF Record Validation – Identifies SPF records to verify email sender policies.
  • DMARC Record Detection – Retrieves DMARC policies for email security compliance.
  • Real-time Input Handling – Accepts domain names through standard input and provides instant verification.

    • Technology Stack:
  • Golang – Core language for building the tool
  • net package – Performs DNS lookups
  • bufio, os, log – Handles user input and logging
