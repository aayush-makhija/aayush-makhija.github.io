Interactive Journal

Golang

The Interactive Journal is a personal diary application that leverages AI to help users explore and understand their emotions. Rather than just storing text, the app uses sentiment analysis to interpret journal entries and respond with helpful insights — encouraging self-reflection and mental clarity.

Key Features:

  • Freeform Journaling: Users can write openly, as they would in a traditional diary.
  • Sentiment Analysis via Logistic Regression: A machine learning model processes the input to detect sentiment (positive, neutral, negative) with an accuracy of 77.8%.
  • Feedback Loop: Based on the sentiment, the app offers reflective prompts or supportive messages to help users think deeper about what they've written.
  • Privacy-First: The application doesn’t store or personalize data, respecting the user’s anonymity and privacy.
  • Prototype in Jupyter Notebook: The initial version is built in Jupyter Notebook, allowing quick iterations and experimentation.

    • Technology Stack:
  • Python for development
  • Scikit-learn for logistic regression implementation
  • NLTK & spaCy for text preprocessing and tokenization
  • Accuracy: Achieved a 77.8% accuracy using logistic regression on a cleaned sentiment dataset
  • Model Pipeline: Includes text normalization, stop word removal, vectorization (TF-IDF), and binary classification

    • I’ve always been interested in how technology can support emotional well-being. This project merges natural language processing with mental health awareness, creating a tool that feels personal while staying private. It also gave me hands-on experience with machine learning — from data preprocessing to training and evaluation.
