Playlist Management System

Golang and PostgreSQL

This project is a Music Playlist Management System built using Go (Golang) and PostgreSQL. It allows users to efficiently manage their music collection by creating albums, artists, playlists, and songs while offering seamless database interactions.

Key Features:

  • Album & Artist Management: Add albums and artists to the database.
  • Song Management: Insert songs with details like name, artist, and album.
  • Playlist Functionality: Create and delete playlists. Add songs to specific playlists. Remove songs from existing playlists.
  • PostgreSQL Integration: Uses database/sql with lib/pq for efficient data handling.

  • CLI-Based User Interaction: Allows users to input selections for song and playlist modifications.
